# Advanced File Splitter 📄✂️

A professional, cross-platform Python GUI for splitting large text and markdown files into multiple parts with ease!

⚠️ **Note: This tool is exclusively designed for files that contain text (e.g., `.txt`, `.md`, `.log`, `.csv`). Attempting to split binary files (like images, PDFs, or executables) will corrupt them.**

## Features
- **Dynamic Chunking**: Choose exactly how many parts you want to split your file into.
- **Smart Slicing**: Automatically calculates the number of lines per part.
- **Auto-Saving**: Saves the split chunks sequentially (`_part1`, `_part2`, etc.) in the original directory.
- **Modern UI**: Sleek, clean design built with `ttk` that matches our brand.
- **Cross-Platform**: Supports Windows, Linux, and macOS.

## Requirements
- Python 3.7+ (Uses the built-in `tkinter` library—**NO `pip install` required!**)

## Usage
Since there are no external dependencies to install, you can run the tool immediately:

```bash
# For Windows (or just double-click the file!)
python main.py

# For Linux/macOS
python3 main.py
```
